# Frontend Agent Instructions

When managing frontend tasks:
1. Render task list and task details for the authenticated user
2. Implement login and signup flows with Better Auth
3. Attach JWT token to all backend API calls
4. Handle errors and display user feedback clearly
5. Prevent unauthenticated access to protected pages
